## 写作原则

- **记解决方案，不记流水账** — 问题是什么、怎么解决的、为什么这样解决
- **没有有价值的东西就不写** — 质量 > 数量
- **写给未来失忆的自己** — 假设你什么都不记得，能看懂吗？
- **附上关键命令和配置** — 复制粘贴就能用
